16-Iterators(迭代器)

List Iterator

  • 进入迭代：x = iter( [list] )
  • 下一个值：next ( t )
  • 输出剩余值：list ( t )
s=[ 3, 4, 5 ]
t = iter(s) #t进入迭代,t=3
next(t) #t=4
next(t) #t=5

u = iter(s) #新的迭代,u=3

Dic Iterator

  • 迭代键：k = iter( d.keys( ) ) # or iter( d )
  • 迭代值：v = iter( d.values( ) )
  • 迭代键值对：i = iter( d.items( ) )

PS：迭代期间字典大小不能发生变化（可以改变值）

‘For’ Statement

  • for 语句同样推动迭代器迭代
  • 被for/list 过的迭代器将失效
ri = iter( range(3,6) )
for r in ri:
    print(r) #ri 失效
