21-Representation

Represatation

  • ‘str’：人类解释语言
  • ‘repr’：python解释语言
  • ‘eval’：返回传入字符串表达式的结果
half = Fraction(1, 2)
repr(half) # Fraction(1, 2)
str(half) # 1/2

repr(half) => half.__repr__()
str(half) => half.__str__()

22-Composition

Link List

s = Link(1, Link(2, Link(3)))
s.first = 5
t = s.rest
t.rest = s  #(5,2)循环
s.rest.rest.rest.rest.rest.first == 2


def add(s, v)
    if s.first>v:
        s.first, s.rest = v, Link(s.first, s.rest)
    elif s.first<v and empty(s.rest):
        s.rest = Link(v)
    elif s.first<v:
        add(s.rest, v)
    return s
